LukeBot

LukeBot é um bot criado para a Twitch, com objetivo de se tornar um bot de fácil manuseio para o streamer, de forma que não seja necessário configurações complicadas, apenas uma configuração inicial, podendo também criar mini-games para o chat da Twitch, entre outros.
